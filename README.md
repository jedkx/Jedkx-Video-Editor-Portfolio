## 🏃‍♂️ Quick Start

```bash
# Clone repository
git clone https://github.com/jedkx/Jedkx-Video-Editor-Portfolio.git
cd Jedkx-Video-Editor-Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit: http://localhost:5173

## 🚀 Deployment

### Automatic GitHub Pages Deployment

```bash
# Deploy to GitHub Pages
npm run deploy
```

Your site will be live at: **https://jedkx.github.io/Jedkx-Video-Editor-Portfolio**

### Manual Deployment Steps

1. **Enable GitHub Pages**:
   - Go to repository **Settings → Pages**
   - Source: **Deploy from branch**
   - Branch: **gh-pages**
   - Click **Save**

2. **Deploy**: Run `npm run deploy`

> Takes 2-3 minutes to deploy after running the command

## 🎨 Customization

### Personal Information
Update content in these components:
- `src/components/HeroSection.tsx` - Name, tagline, CTA buttons
- `src/components/AboutSection.tsx` - Experience, services, background
- `src/components/SkillsSection.tsx` - Technical skills and proficiency
- `src/components/ProjectsSection.tsx` - Video projects and portfolio
- `src/components/ContactSection.tsx` - Contact information and social links

### Styling & Branding
- **Colors**: Edit Tailwind configuration in `tailwind.config.js`
- **Images**: Add project images to `/public/projects/`
- **Typography**: Modify font settings in `src/index.css`
